## Cross-Phase Integration

### Wired Flows (6/7 working)

1. **Content Reporting (SAFE-01):** FeedCard/CheckInDetailScreen -> ReportBottomSheet -> ReportRepository `/reports` -> POST /api/reports -> ReportService -> DB insert + moderationQueue enqueue
2. **User Blocking (SAFE-04):** UserProfileScreen -> BlockRepository `/blocks/$userId/block` -> POST /api/blocks/:userId/block -> BlockService -> DB; bilateral filter in FeedService, CheckinQueryService, DiscoveryService
3. **Moderation Enforcement (SAFE-02, SAFE-03):** Report -> moderationQueue -> SafeSearch -> autoHideContent writes is_hidden=true -> Phase 9.1 filters enforce in all 7 services -> cache invalidation on admin actions
4. **Onboarding to Recommendations (ONBD-01, ONBD-02):** OnboardingScreen 3-page carousel -> GenrePickerScreen -> local save -> sync on auth -> POST /api/onboarding/genres -> DiscoveryService UNION ALL user_genre_preferences
5. **Share Card Pipeline (SHARE-01-04):** CelebrationScreen/BadgeCollection -> ShareRepository -> POST /api/share/checkin/:id or /badge/:id -> satori + resvg -> R2 upload -> SocialShareService OS share sheet; public landing pages at /share/c/:id and /share/b/:id
6. **RSVP + Friends Going (EVENT-01, EVENT-02):** RsvpButton -> RsvpRepository `/rsvp/$eventId` -> POST /api/rsvp/:eventId -> RsvpService toggle; EventDetailScreen -> `/events/$eventId` -> event data; FriendsGoingWidget -> `/rsvp/$eventId/friends` -> avatars + count

### Broken Flow (1/7)

7. **Password Reset (SAFE-05):** ForgotPasswordScreen -> `dioClient.post('/api/auth/forgot-password')` -> resolves to `/api/api/auth/forgot-password` -> **404**. Even if fixed: EmailService sends `soundcheck://reset-password?token=X` link -> **no deep link handler** in app_router.dart. Flow broken at both request and completion stages.

### Pre-existing Broken Path (not a milestone requirement)

- `discover_providers.dart:137`: `/api/search/users` -> `/api/api/search/users` -> 404 (user search in discover screen)
