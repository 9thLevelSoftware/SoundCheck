    **DiscoveryService modification:**
    Modify the `user_genres` CTE in `computeRecommendations()` to UNION with user_genre_preferences for cold-start:
    ```sql
    WITH user_genres AS (
      SELECT genre, SUM(genre_count) as genre_count FROM (
        SELECT b.genre, COUNT(*) as genre_count
        FROM checkins c JOIN events e ON c.event_id = e.id
        JOIN event_lineup el ON e.id = el.event_id
        JOIN bands b ON el.band_id = b.id
        WHERE c.user_id = $1 AND b.genre IS NOT NULL
        GROUP BY b.genre
        UNION ALL
        SELECT genre, 1 as genre_count
        FROM user_genre_preferences
        WHERE user_id = $1
      ) combined
      GROUP BY genre ORDER BY genre_count DESC LIMIT 5
    )
    ```
    This ensures new users with 0 check-ins but genre preferences get personalized recommendations.
