Backend Block API (from RESEARCH.md):
- POST /blocks/:userId/block (DioClient path: '/blocks/$userId/block')
- DELETE /blocks/:userId/block (unblock)
- GET /blocks/:userId/status -> { data: { blocked: boolean } }
- GET /blocks -> { data: [blocks] } (list all blocked users)
- Rate limit: 30 actions/15 minutes
- Self-block: 400 error ("Cannot block yourself")
- Bilateral: isBlocked returns true if EITHER user blocked the other
- Side effects: Blocking auto-unfollows in both directions (server-side)

- `isBlocked(String userId)` -> GET `/blocks/$userId/status`, return `response.data['data']['blocked'] as bool`

- `blockedUsersProvider`: `FutureProvider.autoDispose<List<Map<String, dynamic>>>` that calls `blockRepository.getBlockedUsers()`
- Import dioClientProvider from `../../../../core/providers/providers.dart`
