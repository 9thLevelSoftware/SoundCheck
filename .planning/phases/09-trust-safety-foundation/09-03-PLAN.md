**BlockService.ts:**
- Class with constructor taking optional `db` parameter (default `Database.getInstance()`).
- `blockUser(blockerId: string, blockedId: string): Promise<UserBlock>` — Validate blockerId !== blockedId (throw error if same). INSERT INTO user_blocks. Catch unique violation (already blocked — return idempotent success). Also unfollow in both directions if applicable: DELETE FROM followers WHERE (follower_id = $1 AND following_id = $2) OR (follower_id = $2 AND following_id = $1). Return created block.
- `unblockUser(blockerId: string, blockedId: string): Promise<void>` — DELETE FROM user_blocks WHERE blocker_id = $1 AND blocked_id = $2. If no rows affected, throw "Not blocked".
- `isBlocked(userA: string, userB: string): Promise<boolean>` — SELECT 1 FROM user_blocks WHERE (blocker_id = $1 AND blocked_id = $2) OR (blocker_id = $2 AND blocked_id = $1) LIMIT 1.
- `getBlockedUsers(userId: string): Promise<UserBlock[]>` — SELECT * FROM user_blocks WHERE blocker_id = $1 ORDER BY created_at DESC.
- **CRITICAL: `getBlockFilterSQL(userId: string, userColumn: string): string`** — Returns a SQL fragment: `AND NOT EXISTS (SELECT 1 FROM user_blocks WHERE (blocker_id = '${userId}' AND blocked_id = ${userColumn}) OR (blocker_id = ${userColumn} AND blocked_id = '${userId}'))`. This is the centralized block filter used by ALL content-serving queries.

  NOTE: For the SQL fragment, use parameterized approach where possible. If parameterization is too complex for the existing query builders, use the fragment approach but ensure the userId is validated as a UUID before interpolation.

**BlockController.ts:**
- Class following existing controller pattern.
- `blockUser = async (req, res)` — Extract blockerId from req.user.id, blockedId from req.params.userId. Call blockService.blockUser(). Return 201.
- `unblockUser = async (req, res)` — Call blockService.unblockUser(). Return 200.
- `getBlockedUsers = async (req, res)` — Call blockService.getBlockedUsers(). Return 200 with array.
- `checkBlocked = async (req, res)` — Call blockService.isBlocked(). Return 200 with { blocked: boolean }.

**blockRoutes.ts:**
- Router with:
  - POST '/:userId/block' — authenticateToken, blockController.blockUser
  - DELETE '/:userId/block' — authenticateToken, blockController.unblockUser
  - GET '/' — authenticateToken, blockController.getBlockedUsers
  - GET '/:userId/status' — authenticateToken, blockController.checkBlocked

1. Import or instantiate BlockService
