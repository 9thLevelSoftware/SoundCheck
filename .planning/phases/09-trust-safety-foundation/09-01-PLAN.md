- Create `reports` table: id (UUID PK), reporter_id (FK users ON DELETE CASCADE), content_type (content_type_enum), content_id (UUID), target_user_id (FK users nullable), reason (report_reason), description (TEXT nullable), status (report_status DEFAULT 'pending'), reviewed_by (FK users nullable), reviewed_at (TIMESTAMPTZ), review_notes (TEXT), created_at (TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP). **Migration 027 — User blocks:**
- Create `user_blocks` table: id (UUID PK), blocker_id (FK users ON DELETE CASCADE), blocked_id (FK users ON DELETE CASCADE), created_at (TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP). Add UNIQUE(blocker_id, blocked_id) and CHECK(blocker_id != blocked_id).
- Create indexes: idx_user_blocks_blocker(blocker_id), idx_user_blocks_blocked(blocked_id).

- Create indexes: idx_password_reset_tokens_hash(token_hash), idx_password_reset_tokens_user(user_id).

**Migration 029 — Add is_admin column:**
- ALTER TABLE users ADD COLUMN IF NOT EXISTS is_admin BOOLEAN DEFAULT FALSE.
